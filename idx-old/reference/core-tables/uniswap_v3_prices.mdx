| Column Name              | Data Type  |

| block\_timestamp         | Long       |
| token0                   | String     |
| token0\_name             | String     |
| token0\_symbol           | String     |
| token0\_dec              | BigDecimal |
| token1                   | String     |
| token1\_name             | String     |
| token1\_symbol           | String     |
| token1\_dec              | BigDecimal |
| zero\_to\_one            | BigDecimal |
