## Example Requests

Use `chain_ids=42170` to query Arbitrum Nova specifically.

### Balances

<CodeGroup>

```bash cURL
curl -X GET "https://api.sim.dune.com/v1/evm/balances/0xd8dA6BF26964aF9D7eEd9e03E53415D37aA96045?chain_ids=42170" \
  -H "X-Sim-Api-Key: YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch("https://api.sim.dune.com/v1/evm/balances/0xd8dA6BF26964aF9D7eEd9e03E53415D37aA96045?chain_ids=42170", {
  headers: { "X-Sim-Api-Key": "YOUR_API_KEY" }
});
const data = await response.json();
```

```python Python
import requests

response = requests.get(
    "https://api.sim.dune.com/v1/evm/balances/0xd8dA6BF26964aF9D7eEd9e03E53415D37aA96045?chain_ids=42170",
    headers={"X-Sim-Api-Key": "YOUR_API_KEY"}
)
data = response.json()
```

</CodeGroup>
