# Transaction Context

* [simCalldata](/reference/simfunctions/simcalldata): Returns the calldata of the (internal) transaction
* [simCallDepth](/reference/simfunctions/simcalldepth): Returns the call depth of the currently executing (internal) transaction
* [simCallKind](/reference/simfunctions/simcallkind): Returns the call kind of the (internal) transaction
* [simFrom](/reference/simfunctions/simfrom): Returns the from address of the (internal) transaction
* [simGas](/reference/simfunctions/simgas): Returns the amount of gas assigned to this (internal) transaction
* [simGasLeft](/reference/simfunctions/simgasleft): Returns the gas remaining after an (internal) transaction
* [simTo](/reference/simfunctions/simto): Returns the to address of the (internal) transaction
* [simValue](/reference/simfunctions/simvalue): Returns the value sent with the (internal) transaction
* [simTransactionGasLimit](/reference/simfunctions/simtransactiongaslimit): Returns the gas limit of the transaction
* [simTransactionHash](/reference/simfunctions/simtransactionhash): Returns the transaction hash
* [simTransactionIndex](/reference/simfunctions/simtransactionindex): Returns the index of the transaction within its block
* [simTransactionStatus](/reference/simfunctions/simtransactionstatus): Returns the transaction status
* [simCodeAddress](/reference/simfunctions/simcodeaddress): Returns the code address of the (internal) transaction
* [simCreatedContract](/reference/simfunctions/simcreatedcontract): Returns the gas remaining after an (internal) transaction
* [simImplementsFunction](/reference/simfunctions/simimplementsfunction): Checks if a contract implements a function
* [simParentCodeAddress](/reference/simfunctions/simparentcodeaddress): Returns the parent code address of the (internal) transaction
* [simSalt](/reference/simfunctions/simsalt): Returns the salt of a CREATE2 transaction

# Storage and State

* [simReadStorage](/reference/simfunctions/simreadstorage): Reads storage for specified (contract, slot)
* [simSetStorage](/reference/simfunctions/simsetstorage): Overwrites the contents of any (contract, slot)
* [simStorageAddress](/reference/simfunctions/simstorageaddress): Returns the address whose slot was read/written
* [simStorageKey](/reference/simfunctions/simstoragekey): Returns the key of the slot that was read/written
* [simStorageValue](/reference/simfunctions/simstoragevalue): Returns the value of the slot that was read/written
* [simSetBalance](/reference/simfunctions/simsetbalance): Overwrite the native token balance of a given address

# Logs and Events

* [simEmitsLog](/reference/simfunctions/simemitslog): Checks if a contract emits a log
* [simLogAddress](/reference/simfunctions/simlogaddress): Get the address that emitted a log
* [simLogData](/reference/simfunctions/simlogdata): Returns the unindexed event data of a log
* [simLogTopic](/reference/simfunctions/simlogtopic): Returns a topic from a log
* [simLogTopicSize](/reference/simfunctions/simlogtopicsize): Returns the number of topics in a log

# Utilities

* [simAddressNonce](/reference/simfunctions/simaddressnonce): Returns the nonce of an address
* [simToString](/reference/simfunctions/simtostring): Converts many data types to string for easy logging
