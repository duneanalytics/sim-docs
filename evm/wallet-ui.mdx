## The Wallet UI We'll Build

<Frame caption="The final UI we'll build together in this guide">
    <img src="/images/wallet-ui.png" />
</Frame>

We'll create a wallet dashboard with three main features:
1. Token balances with real-time USD values across all EVM chains
2. Recent wallet activity showing transfers and interactions
3. NFT collection display with metadata and images

Let's start by setting up our project:

```bash
# Create project directory
mkdir wallet-ui
cd wallet-ui

# Initialize project
npm init -y

# Install dependencies
npm install express ejs dotenv

# Create necessary files
touch index.js
cp .env.template .env  # We'll create this template next
```

Create a `.env.template` file:

```plaintext:.env.template
# Required - Get your API key from dune.com
DUNE_API_KEY=your_api_key_here

# Optional - defaults to 3000
PORT=3000
```

And set up the basic project structure:

```bash
wallet-ui/
├── index.js           # Main application file
├── package.json       # Project configuration
├── .env              # Your environment variables (don't commit!)
├── .env.template     # Template for environment variables
├── public/           # Static assets
│   └── css/
│       └── style.css
└── views/            # EJS templates
    └── index.ejs
```

This scaffold provides everything we need to start building our wallet interface. In the following sections, we'll implement each feature step by step, starting with token balances.
