The Collectibles API provides information about NFTs (ERC721 and ERC1155 tokens) owned by a specific address on supported EVM blockchains.

<SupportedChains endpoint="collectibles" />

## Usage

<RequestExample>

```bash cURL
curl -X GET "https://api.sim.dune.com/v1/evm/collectibles/0xd8dA6BF26964aF9D7eEd9e03E53415D37aA96045?chain_ids=mainnet&limit=100" \
  -H "X-Sim-Api-Key: YOUR_API_KEY"
```

```javascript JavaScript
const res = await fetch("https://api.sim.dune.com/v1/evm/collectibles/0xd8dA6BF26964aF9D7eEd9e03E53415D37aA96045?chain_ids=mainnet&limit=100", {
  headers: { "X-Sim-Api-Key": "YOUR_API_KEY" }
});
const data = await res.json();
```

```python Python
import requests

url = "https://api.sim.dune.com/v1/evm/collectibles/0xd8dA6BF26964aF9D7eEd9e03E53415D37aA96045"
headers = {"X-Sim-Api-Key": "YOUR_API_KEY"}
params = {"chain_ids": "mainnet", "limit": 100}
response = requests.get(url, headers=headers, params=params)
print(response.json())
```
